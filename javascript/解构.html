    比如在我们日常的使用中数组解构我们就是用数组来接收，对象解构就是用对象解构。

    # 数组解构

    只要是某种数据可以进行迭代的，我们都可以使用数组来解构。

    1. 数组解构 这个也是我们日常生活中使用比较多的。
    const [a, b] = [1, 2];
console.log(a, b); // a=1 b=2
    1. Set 数据解构
    const [a, b, c] = new Set([1, 2, 2]);
console.log(a, b, c); // a=1 b=2 c=undefined

    为什么这个时候 c !== 2?

    原因在于 set 对象只存储唯一值，这也是我们会用 set 来去重的原理

    Set 对象允许你存储任何类型（无论是原始值还是对象引用）的唯一值

    // Set和解构结合使用
const arr = [1, 2, 2, 3, 5];
const uniqueArr = [...new Set(arr)];
console.log(uniqueArr); // [1, 2, 3, 5]
